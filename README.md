# Project Overview
This project describes the complete machine learning project which focuses on regression problem where predicting the sales prices of residential properties in King County, Washington, USA. It contains all the necessary components for the project including data collection, data pre-processing, exploratory data analysis, feature selection, model building, optimization, evaluation and business insights. The process and reasons for making decisions during the analysis are also described in the sections.
